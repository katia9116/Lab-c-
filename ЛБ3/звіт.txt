3.1. Що таке умовний оператор?
Умовний оператор — це конструкція, яка забезпечує виконання певних дій залежно від істинності або хибності логічної умови.

3.2. Які є умовні оператори?
if
if…else
if…else if…else
switch

3.3. Яка структура switch?
switch (вираз) {
    case значення1:
        оператори;
        break;
    case значення2:
        оператори;
        break;
    ...
    default:
        оператори;
}

3.4. Яка форма у тернарного оператора?
умова ? вираз1 : вираз2;
